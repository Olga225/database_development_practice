Постановка задачи:

Определить сумму цифр в произвольной символьной строке. Если результат больше 10, то процесс продолжить — снова найти сумму цифр и так далее — пока не получится одна цифра.

Объяснение алгоритма работы кода:

Для начала необходимо выделить из строки число, то есть убрать все нецифровые символы.
Далее эту задачу можно решить, циклично обрабатывая число как строку, суммируя все ее цифры по одной. Если итоговая сумма меньше 10, то цикл завершается, иначе за обрабатываемое число принимается число получившейся суммы, и для него повторяется тот же алгоритм.
Для достижения подобного результата средствами MODEL надо использовать несколько вспомогательных столбцов. 
1.	Каждая строка должна быть уникально обозначена для возможности обращаться к нужным предыдущим строкам
2.	Должно осуществляется хранение обрабатываемого сейчас числа и цифр, которые прибавляются к сумме на каждой итерации внутри одного цикла
3.	Должна осуществляться проверка, что обрабатываемое число было рассмотрено до конца
4.	Должен быть «буфер», позволяющий алгоритму понимать, из какой строки начинается новый цикл
5.	Должна осуществляться проверка достижения условия задачи, т.е. получение итоговой суммы всех цифр числа, меньшей 10.
По результатам нескольких выполнений цикла должна быть получена и выведена итоговая сумма.
